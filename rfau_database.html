Real-classroom Facial Action Unit (RFAU) Database

Details

Real-classroom Facial Action Unit (RFAU) Database is a manually-annotated facial action unit database from 1,796 juveniles with 256,220 facial images in real classrooms, covering different subjects and grades in primary and high schools. Each image has been labeled with 12 action units of 6-level intensity by certificated Facial Action Coding System (FACS) annotators. This database covers many challenges in the wild, including different head poses, facial resolution, illumination, and occlusion. RFAU-DB has large diversities, large quantities, and rich annotations, including:

  • 256,220 number of real-classroom facial images from 1,796 juveniles in 275 courses,

  • 12 action units annotation of 6-level intensity for each image,

  • 68 landmark locations, bounding box, grade and subject annotations

  • baseline : 6 state-of-the-art algorithms for facial aciton unit detection and 2 state-of-the-art algorithms for facial aciton unit intensrty estimation

To be able to objectively measure the performance of the algorithms, the database has been split into 3 folds. 3-fold cross validation is recommend.

How to download the Database


This database is publicly available. It is free for professors and researcher scientists affiliated to a University.
Permission to use but not reproduce or distribute the RFAU database is granted to all researchers given that the following steps are properly followed:
  • Send an e-mail to Fei Jiang (jiangf@sjtu.edu.cn) and Qiaoping Hu (huqiaoping23@sjtu.edu.cn) to get a license agreement. Your Email MUST be set from a valid University account.
  • Sign the license agreement. The LA must be signed by your group leader or professor (who has fixed position in a University).
  • An email with the download link and password will be sent to you. You must download in one week.
